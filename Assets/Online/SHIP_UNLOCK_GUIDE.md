## Overview

**Total Ships**: 17 ships (excluding starter ship)
- Starter Ship: **Vanguard** (given at account creation)
- Unlockable Ships: **17 ships** from Level 3 to Level 50

**Ship Distribution by Class**:
- **All-Around**: 5 ships (balanced, beginner-friendly)
- **Tank**: 3 ships (high armor, slow)
- **Damage Dealer**: 4 ships (glass cannon)
- **Controller**: 5 ships (utility, crowd control)

## Unlock Timing Strategy

### Why This Distribution?

**Early Game (1-10)**: 4 ships
- Unlock every 3-4 levels
- Focus on All-Around and first Tank
- **Goal**: Keep new players engaged without overwhelming

**Mid Game (11-25)**: 6 ships
- Unlock every 2-4 levels
- Introduce DD class and more variety
- **Goal**: Maintain steady progression, introduce all classes

**Late Game (26-40)**: 5 ships
- Unlock every 2-4 levels
- Focus on Controller class and milestones
- **Goal**: Reward dedicated players, complete all class rosters

**Prestige (41-50)**: 2 ships
- Unlock every 5 levels
- Best-in-class ships
- **Goal**: Ultimate rewards for max-level players

### Milestone Ships

Special ships tied to major progression milestones:

| Level | Milestone | Ship Reward |
|-------|-----------|-------------|
| 10 | Ranked Unlocked | Eclipse Striker (All-Around) |
| 20 | Custom Slot #2 | Horizon Vanguard (All-Around) |
| 30 | Veteran Status | Crimson Tempest (DD, Legendary) |
| 40 | Custom Slot #3 | Infinity Class (Controller) |
| 50 | Max Level | Celestial Monarch (Controller, Ultimate) |

## Class Balance

Each class gets ships at appropriate pacing:

**All-Around** (5 ships):
- Most ships because it's the beginner class
- Spread across all level ranges
- Always a fallback option

**Tank** (3 ships):
- Fewer ships since it's specialized
- Unlocks early (Level 6-18)
- One elite late-game option (35)

**Damage Dealer** (4 ships):
- Glass cannon playstyle
- Unlocks mid-game (Level 16-30)
- Includes legendary ship at 30

**Controller** (5 ships):
- Advanced playstyle
- Unlocks late (Level 26+)
- Multiple ships since it's most complex
- Includes ultimate ship at 50

## Customization Guide

### How to Add New Ships

Edit `ProgressionSystem.cs`:

```csharp
private static readonly Dictionary<int, ShipUnlockData> SHIP_UNLOCKS = new Dictionary<int, ShipUnlockData>
{
    // Add your new ship here:
    { 15, new ShipUnlockData("my_new_ship", "My Ship Name", ShipClass.Tank, "Ship description") },
};
```

### How to Change Unlock Levels

```csharp
// Move Nova Class from Level 3 to Level 5:
{ 5, new ShipUnlockData("nova_class", "Nova Class", ShipClass.AllAround, "...") },
```

### How to Remove Ships

Simply delete the entry from the dictionary:

```csharp
// Remove this line to remove the ship:
// { 7, new ShipUnlockData("phoenix_mk1", "Phoenix Mk-I", ...) },
```

### How to Change Ship Names

```csharp
// Change display name (keeps same ID for game logic):
{ 3, new ShipUnlockData("nova_class", "Super Nova", ShipClass.AllAround, "New description") },
```

## Testing Ship Unlocks

### Debug Commands (Add to your debug menu)

```csharp
// Unlock a specific ship for testing
public static void DEBUG_UnlockShip(PlayerProfileData profile, string shipId)
{
    if (!profile.unlockedShipModels.Contains(shipId))
    {
        profile.unlockedShipModels.Add(shipId);
        Debug.Log($"DEBUG: Unlocked ship {shipId}");
    }
}

// Unlock all ships
public static void DEBUG_UnlockAllShips(PlayerProfileData profile)
{
    foreach (var ship in SHIP_UNLOCKS.Values)
    {
        if (!profile.unlockedShipModels.Contains(ship.shipId))
        {
            profile.unlockedShipModels.Add(ship.shipId);
        }
    }
    Debug.Log("DEBUG: All ships unlocked");
}

// Get next ship unlock info
public static void DEBUG_ShowNextShip(int currentLevel)
{
    int nextLevel = GetNextShipUnlockLevel(currentLevel);
    if (nextLevel != -1)
    {
        var ship = GetShipUnlock(nextLevel);
        Debug.Log($"Next ship: {ship.displayName} at Level {nextLevel}");
    }
}
```

## Balancing Tips

### Unlock Frequency

**Current**: Average 1 ship every 2.7 levels
- Too fast: <2 levels between ships
- Too slow: >5 levels between ships
- Sweet spot: 2-4 levels

### Class Introduction

- All-Around: Available from start ‚úì
- Tank: Unlocked at 5, first ship at 6 ‚úì
- DD: Unlocked at 15, first ship at 16 ‚úì
- Controller: Unlocked at 25, first ship at 26 ‚úì

**Rule**: First ship of each class should appear **1 level after class unlocks**

### Milestone Rewards

Every major milestone (10, 20, 30, 40, 50) should have a ship unlock to feel special.
